## How to Use These Bullets:

### For Resume:

- Select 2-3 most relevant bullet points per project
- Adjust language based on job description keywords
- Focus on bullets that highlight skills required for the position
- Include metrics/results when available

### For Portfolio Website:

- Use the full set of bullets as project descriptions
- Organize them by feature/achievement sections
- Include before/after if applicable
- Add links to live demos and GitHub repositories

### For Interviews:

- Prepare talking points for each bullet
- Have examples ready for technical questions
- Focus on your personal contributions in team projects
- Highlight problems you solved and impact you made
